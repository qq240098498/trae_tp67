## 2. 核心业务流程图

```
团长登录系统
      │
      ▼
┌─────────────────────┐
│  【仪表盘】数据概览  │  ← 首页入口、待办提醒
└─────────┬───────────┘
          │
          ▼
┌─────────────────────┐
│  【团品管理】发布团品 │  ← 自定义团品 / 供应链选品
└─────────┬───────────┘
          │
          ▼
┌─────────────────────┐
│ 团员下单（模拟数据）│  ← 自动生成订单、取货码
└─────────┬───────────┘
          │
          ▼
┌─────────────────────┐
│ 【下单统计】数据统计 │  ← 按团品/团员维度统计
└─────────┬───────────┘
          │
          ▼
┌─────────────────────┐
│ 【到货分拣】分拣打印 │  ← 分拣签/取货标签打印
└─────────┬───────────┘
          │
          ├──────────────────────────────┐
          │                              │
          ▼                              ▼
┌─────────────────────┐       ┌─────────────────────┐
│ 【取货核销】扫码取货 │       │【超时催收】提醒代存  │
└─────────┬───────────┘       └─────────┬───────────┘
          │                              │
          ▼                              ▼
┌─────────────────────┐       ┌─────────────────────┐
│    订单正常完成     │       │   代存 / 退货处理   │
└─────────┬───────────┘       └─────────────────────┘
          │
          ├──────── 无售后 ────────► 流程结束
          │
          ▼ 有售后
┌─────────────────────┐
│ 【售后登记】退款处理 │  ← 缺货/破损/质量问题
└─────────┬───────────┘
          │
          ▼
    流程闭环完成
```
